📖 사용 가이드

1새 기록 만들기
  • 목록 화면 오른쪽 상단 ＋ 새 기록 버튼을 누르세요.
  • 그룹공부마다 하나의 기록을 만들면 돼요.
  • 입력한 내용은 자동으로 저장됩니다.
💡 앱을 닫았다 열어도 데이터가 그대로 유지돼요.
2① 시작 보고서 작성
  • 봉사자 기본정보를 입력하세요 (공부하는 곳, 요일·시간, 주제, 봉사자 이름 등).
  • 그룹원 명단은 ＋ 추가 버튼으로 한 명씩 추가해요.
  • 작성 완료 후 🖨️ 시작 보고서 PDF를 눌러 제출하세요.
※ 그룹 형성 즉시 ARCA 센터에 제출
Tel. 051) 519-0483 / bsarca1@gmail.com
3② 출석부 입력
  • 날짜 입력란에 1과 9/9 형식으로 입력 후 ＋ 추가를 눌러요.
  • 추가된 날짜를 탭하면 수정하거나 삭제할 수 있어요.
  • 각 그룹원의 출결을 아래에서 선택하세요:
○ × △
  • ○ 출석   × 결석   △ 지각/조퇴
⚠️ 날짜를 삭제하면 해당 날짜의 출결 데이터도 함께 삭제돼요.
4③ 마침 보고서 작성
  • 마침일을 입력하면 총 공부 기간이 자동 계산돼요.
  • 각 그룹원의 연수 차수를 입력하세요.
  • 특기사항과 소감을 작성하고 🖨️ 마침 보고서 PDF로 저장하세요.
📋 그룹공부 기록과 연수사항 명부 기록은 센터에서 작성해요. PDF에 빈칸으로 출력됩니다.
5백업 & 불러오기
  • 💾 백업 (내보내기) — 모든 기록을 .json 파일로 저장해요. 핸드폰 교체 전이나 앱 초기화 전에 꼭 백업하세요.
  • 📂 불러오기 — 저장해 둔 .json 파일을 선택하면 기록이 복원돼요.
⚠️ 불러오기를 하면 기존 데이터가 덮어써집니다. 먼저 백업을 권장해요.
6PDF 저장 방법
  • 각 탭 상단의 🖨️ PDF 버튼을 누르면 새 탭이 열려요.
  • 새 탭에서 「인쇄 / PDF 저장」 버튼을 누르세요.
  • 프린터 선택 화면에서 「PDF로 저장」을 선택하면 완료!
📱 모바일에서는 공유 → 「파일에 저장」 또는 「프린트」로 PDF를 만들 수 있어요.
